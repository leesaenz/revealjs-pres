Ember CLI and HighCharts

For my presentation, I chose Highcharts.js, built inside of Ember CLI.

What is Highcharts?

Highcharts is a program for creating responsive charts, easily within a framework or without. Highcharts supports many different kinds of charts, including pie, bar, line, and column. The example I made uses column and line.

Setting up Highcharts in Ember

Setting up Highcharts in Ember is quite simple. There are a few options for this. The first is to use npm to install ember-highcharts and generate charts via CLI. The second, which I did for simplicity, is to create controllers and add your chart options there.
